## Bonfire
Node job scheduling backed by Firebase RD.

###### GETTING STARTED

###### USING BONFIRE
You can install Bonfire using Yarn or NPM.

```sh
# Via Yarn:
$ yarn add @karn/bonfire --save
# or via NPM:
$ npm install @karn/bonfire --save
```

```javascript
import { Bonfire } from '@karn/bonfire'
```
